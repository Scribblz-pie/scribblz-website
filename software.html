Web Interface

üíª

The web interface allows users to draw or upload images that the robot will recreate on the wall. Built with HTML5 Canvas and JavaScript, it provides real-time feedback and communicates with the robot via WebSocket connections.

Features:

  • Interactive drawing canvas
  • Image upload and processing
  • Real-time robot status updates
  • Drawing preview and path optimization

Raspberry Pi Control Software

üêç

The Raspberry Pi runs Python-based control software that handles:

  • Path Planning: Converts user drawings into optimized motor commands
  • Communication: Manages WebSocket server for web interface
  • State Management: Tracks robot position and drawing progress
  • Error Handling: Detects and recovers from common failure modes

Microcontroller Firmware

The microcontroller firmware is written in C/C++ and handles real-time control tasks:

  • Motor Control: PID loops for precise speed and position control
  • Sensor Processing: Real-time reading and filtering of sensor data
  • Safety Systems: Emergency stop and fault detection
  • Communication Protocol: Serial interface with Raspberry Pi

Control Algorithms

üìê

Several key algorithms enable smooth, accurate drawing:

  • Path Optimization: Minimizes drawing time while maintaining quality
  • Motion Planning: Generates smooth trajectories between waypoints
  • PID Control: Maintains precise motor speeds and positions
  • Localization: Estimates robot position using encoder and IMU data
